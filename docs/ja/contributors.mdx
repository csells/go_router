# コントリビューター

既に本プロジェクトに貢献いただいた方がこんなに多いとは驚きです。
この場をお借りして go_router のコントリビューターに感謝申し上げます。

- [Salakar](https://github.com/Salakar) : GitHub の CI アクション設定（私がいつも忘れがちなことをアシストしてくれる）。
- [rydmike](https://github.com/rydmike) : README およびドキュメントの修正、ネスト型ナビゲーションで状態を保つサンプルコードの提供。
- [Abhishek01039](https://github.com/Abhishek01039) フィールドの後にコンストラクタを書く私の習癖（Dart のベストプラクティスに反する）を直すのを手伝ってくれた。
- [SunlightBro](https://github.com/SunlightBro) Android のシステムバックボタンに関する修正。
- [craiglabenz](https://github.com/craiglabenz) README の修正。
  また、Craig は名前付きルートにおいて型安全な go と push のコードを build_runner で生成する[提案](https://github.com/csells/go_router/issues/66)をしてくれた。
  今後の go_router でこの機能を利用したい方は いいね! を。
- [kevmoo](https://github.com/kevmoo) README におけるスペルミスや未使用インポートの発見、そしてメンテナンス容易性を高めるリファクタリングを手伝ってくれた。
- [andyduke](https://github.com/andyduke) 素晴らしい [`navigatorBuilder`](/navigator-builder) の追加と、
  [状態復元](https://github.com/csells/go_router/blob/master/example/lib/state_restoration.dart)に関するアップデート。
- [lulupointu](https://github.com/lulupointu) ディープリンクに飛ぶと画面が一瞬チラつく問題の修正。
- [jopmiddelkamp](https://github.com/jopmiddelkamp) `refreshListenable` に使用する `GoRouterRefreshStream`（`Stream` を `Listenable` にマッピング）の追加、
  そしてプルリクエストのテンプレート作成。
- [mehade369](https://github.com/mehade369) ドキュメントの修正。
- [ben-milanko](https://github.com/ben-milanko) サンプルリンクの修正。
- [johnpryan](https://github.com/johnpryan) デバッグ診断に logging パッケージを使用するアップデート。
